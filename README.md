## 🌌 The Next Generation of Text Analytics

Welcome to the **Holosphere 3D Sentiment System** – a paradigm shift from traditional flat web applications into an immersive, volumetric spatial dimension. This advanced AI engine processes high-density text arrays and extranet URLs, mapping human emotion, polarity, and subjectivity directly into interactive 3D spatial environments.

Built on **Streamlit** and heavily supercharged with **Custom 3D Cyber-CSS**, the suite utilizes literal Z-axis physics, interactive 3D matrices, and tactile extrusions to construct an advanced NLP experience.

---

## 💎 Breakthrough 3D Features

✅ **TRON-Style Animated Space Grid** - A continuously panning, isometric holographic floor constructed using bleeding-edge CSS grid transformations (`perspective` & `rotateX`).
✅ **Interactive XYZ Volumetric Plotting** - Replaces basic charts with massive interactive 3D scatter topologies (`plotly.express.scatter_3d`), charting sentence timelines, scale, and emotion directly on the Z-axis.
✅ **Physical 3D Tactile Mechanics** - Buttons emulate real mechanical switches in the DOM, computing physical drops and squished borders.
✅ **Holographic Extruded Nodes** - Hovering over dashboard metric cards physically transforms them in 3D-space toward your cursor, dropping heavy neon aesthetic shadows. 
✅ **Global URL Intercept** - Feed the Matrix any URL, and it dynamically rips the text out, maps the neural components, and generates a clustered temporal 3D wave map of the author's emotional shifts!

## 🛠️ Architecture & Tech Stack

**Neural & Spatial Backbone:**
- **Python 3.11+** - The overarching machine logic
- **Streamlit** - Core rendering engine and rapid dashboard logic
- **Plotly Express** - The mathematical framework handling the heavy 3D XYZ topology rendering
- **TextBlob / NLTK** - Robust polarity and entity extraction NLP cores
- **BeautifulSoup4** - High-speed DOM interception for the URL Web Scraper

**Front-End Engine:**
- Deeply injected 3D `CSS` (transform: translateZ, preserve-3d)
- Custom Linear Gradients & Backdrop filters for absolute Glassmorphism

## 💻 Installation & Ignition

### 1. Synchronize the Repository
```bash
git clone https://github.com/HarshChoudhary2003/sentiment-analysis-webapp.git
cd sentiment-analysis-webapp
```

### 2. Install the Required Neural Dependencies
Ensure your local terminal has the massive computational packages required to render this:
```bash
pip install streamlit textblob plotly beautifulsoup4 emoji pandas
```

*Note: The engine features an Auto-Corpus Catcher for NLTK, attempting to download missing models safely at runtime.*

### 3. Ignite the Holosphere
```bash
streamlit run Sentiment.py
```
*Wait for the local web server to bind, then open the dimensional gateway in your browser!*
