## 🌟 Vision & Architecture
Welcome to the next generation of visual linguistics. This application moves away from standard flat data dashboards and employs advanced cascading style techniques to simulate physical depth, refraction, and true 3D spatial mapping for real-world NLP and web scraping tasks. 

<br>

## ✨ Premium Interface Features
- **Volumetric Glass Panels:** Heavy use of `backdrop-filter: blur(20px)` for beautiful translucency across UI components.
- **Dynamic Floating Orbs:** Animated radial gradients float dynamically beneath the glass elements to emphasize physical UI depth and provide a stunning lightshow.
- **Premium Typography & Reflections:** Clean custom font integrations with linear-gradient masked properties on key data metrics (shifting from titanium white to ice-blue).
- **Tactile Interactive States:** Hover effects that physically elevate components and reveal dynamic specular highlights simulating interactive physical glass.
- **Immersive 3D Charting:** Completely transparent-background Plotly 3D scatter charts that merge cleanly into the gradient glass layout.

<br>

## 🚀 Core Modules

### 1. 🧩 Matrix Decoder
Drop high-density text arrays (paragraphs, articles, speeches) into the decoder. The engine refracts data through neural pathways to extract **Polarity**, **Subjectivity**, and core contextual themes. 
* Visually generates a **3D Dimensional Topology Map** projecting lexical lengths against sentiment biases in an interactive 3D spatial void.

### 2. 🌐 URL Quantum Uplink
Input a target web coordinate (URL) and hack the mainframe! The application intercepts the remote extranet code, parsing the text to render a complete **Temporal 3D Document Waveform**. This allows you to visually travel through the storyline's emotional arc from the beginning of the site to the end.

## 🛠️ Installation & Booting

**1. Clone the Central Repository**
```bash
git clone https://github.com/HarshChoudhary2003/sentiment-analysis-webapp.git
cd sentiment-analysis-webapp
```

**2. Establish Neural Links (Dependencies)**
Ensure you have the required modular libraries installed:
```bash
pip install streamlit textblob pandas emoji beautifulsoup4 plotly
```
*(Optional: Textblob requires base NLTK corpora to parse nouns. Run `python -m textblob.download_corpora` if prompted).*

**3. Ignite the Engine**
```bash
streamlit run Sentiment.py
```
*The local dev server will boot and seamlessly map your default browser to `localhost:8501`.*
